Metric Value Description

vendorResolvableOnly 0 (not linked only) Vendor is resolvable, but product is not.
unresolvable 0 (not linked only) Neither vendor nor product can be resolved from normalized strings.
needsNormalization 0 Row has not been normalized (or normalized fields are blank).
