Metric Definitions (click to expand)
checked
Number of SoftwareInstall records evaluated in this view (after applying limit and filters).
linked
Records where both cpe_vendor_id and cpe_product_id are filled in the database. (This is a SQL-level flag; it does not guarantee the referenced dictionary rows still exist.)
linkedValid
Records that are linked in SQL and whose referenced vendor/product entries still exist in the CPE dictionary and are structurally consistent.
linkedStale
Records that are linked in SQL but whose referenced vendor or product entries no longer exist in the CPE dictionary, or are inconsistent.
resolvable
Records whose current vendor/product strings (after normalization and alias resolution) can be matched to an existing vendor and product in the CPE dictionary, regardless of whether SQL-level links (cpe_vendor_id / cpe_product_id) are already set.
unresolvable
Records that cannot be resolved to any vendor/product entry in the current CPE dictionary using normalized strings.
needsNormalization
Records where normalized_vendor or normalized_product is blank or missing, indicating incomplete canonicalization.
