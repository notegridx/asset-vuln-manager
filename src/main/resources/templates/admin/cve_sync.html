CVE Data Update

Download and refresh vulnerability data from the NVD CVE JSON 2.0 feed. This will update stored CVEs and their affected product/version ranges used for matching and alerts.

Notes:
  • Safe for large feeds (streaming + chunked transactions).
  • Idempotent: re-running will not create duplicates.
  • If the selected feed has not changed since the last run, the sync may be skipped unless forced.
  • MODIFIED is typically used for routine updates; RECENT is a smaller feed for quick setup/testing.
