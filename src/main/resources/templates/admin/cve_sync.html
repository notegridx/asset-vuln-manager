Update Vulnerability Data

Download and refresh vulnerability data from the NVD CVE JSON 2.0 feed. This will update stored vulnerabilities and their affected product/version ranges used for alert generation.

Notes:
  • Safe for large feeds (streaming + chunked transactions).
  • Idempotent: re-running will not create duplicates.
  • If the selected feed has not changed since the last run, the update may be skipped unless forced.
  • MODIFIED is typically used for routine updates; RECENT is a smaller feed for quick setup/testing.
