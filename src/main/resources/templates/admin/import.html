Import Inventory

Upload CSV files to register assets and installed software. Installed software is used to match against CVEs and generate alerts.

Tips:
  • Preview only runs validation without saving changes.
  • Apply changes writes to the database.
  • If dictionary validation is enabled, unknown vendor/product pairs may be rejected. You can selectively override DICT_* errors after review.
