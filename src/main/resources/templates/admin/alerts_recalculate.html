Generate Alerts

Analyze installed software against stored vulnerability data and create or update security alerts. This operation only performs matching and alert upsert. It does not import vulnerability data, update the product dictionary, or run canonical backfill.

When to use this:
  • After importing software via CSV or JSON
  • After updating product mappings
  • After updating vulnerability or product dictionary data (if full sync was not used)
