JSON Import

Upload JSON files and import inventory in 3 steps: Upload → Preview → Import. Staging tables are used to keep Preview/Import separated.

JSON Upload Admin Tool
Policy (initial implementation)
  • Import is upsert only (no delete).
  • last_seen_at is updated at import time.
  • source_type is fixed to JSON_UPLOAD.
  • Assets key: externalKey
  • Software key: asset_id + vendor + product + version (DB UNIQUE)
