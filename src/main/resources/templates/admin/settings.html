Canonical Mapping

Applies to: Admin > Canonical Mapping, unresolved mapping review flows, software create/edit screens that use dictionary suggestions, Vulnerability detail candidate suggestions, and software resolve/import/backfill processing. These settings affect how raw values are normalized, how candidates are shown and ranked, how automatic linking behaves, and how the UI explains the result.

Shared Matching Rules

Reserved for future cross-screen rules shared by canonical candidate search, canonical auto-link resolution, and vulnerability unresolved suggestion scoring. No active shared rules are configurable here yet, but this section remains visible so users can see where those controls will live.

Change Control

Save writes the current values to the system settings table immediately. Reset restores default values for all settings on this page. Some saved values already affect active behavior, while others are intentionally stored in advance for future wiring.
